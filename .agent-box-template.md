
## Instanziierungsnachweis

Der Instanziierungs-Sprechakt wird als Markdown-Artefakt belegt:

```text
.agent-box/instantiation.md
```

Eine erneute Instanziierung ist verboten, solange diese Datei existiert,
außer es liegt eine explizite menschliche Freigabe für `--force` vor.
